# Modern Ecommerce Storefront

A sleek, interactive product catalog and shopping cart experience demonstrating modern e-commerce patterns.

## Features

- **Product Catalog:** Dynamic product grid with rich visuals and category organization.
- **Shopping Cart:** Fully functional shopping cart state management (Add, Remove, Adjust Quantity).
- **Persistent Storage:** Cart contents are saved to LocalStorage, preserving the user session.
- **Category Filtering:** Filter products instantly by category without page reloads.
- **Interactive UI:** Smooth transitions and slide-out cart drawer for a seamless user experience.
- **Modern Stack:** Built with Next.js, TypeScript, Tailwind CSS, and Radix UI primitives.
